# Welcome to Toona

Toona is a Matrix-based task collaboration platform that combines instant messaging with kanban-style task management, inspired by Trello.

## Key Features

- **Kanban Board Interface**: Visual task management with drag-and-drop cards
- **Matrix Protocol**: End-to-end encrypted team communication
- **Real-time Collaboration**: Multi-user sync across all platforms
- **Cross-platform**: Runs on macOS, Windows, Linux, Android, and iOS
- **Task Management**: Todo lists, tags, due dates, and more
- **Rich Messaging**: Text, images, replies, reactions, and editing

## Platform Support

Built with Rust and Makepad, Toona runs natively on:
- macOS
- Windows
- Linux
- Android
- iOS

All from a single codebase!
