# Getting Started with Toona

This guide will help you get Toona installed and running on your system.

## Prerequisites

- Rust 1.79 or later
- Git

## Installation

### From Source

```bash
# Clone the repository
git clone https://github.com/toona/toona.git
cd toona

# Build and run
cargo run --release
```

## First Steps

After installation:

1. **Login**: Connect to your Matrix homeserver
2. **Create a Board**: Set up your first kanban board
3. **Add Cards**: Start organizing your tasks
4. **Invite Team**: Collaborate with others
